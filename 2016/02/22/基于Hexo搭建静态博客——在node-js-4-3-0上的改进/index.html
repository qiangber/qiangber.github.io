基于Hexo3.0搭建静态博客

发表于 2016-02-22 | 分类于 Bugs |

Hexo搭建Github静态博客
基于上面这篇文章来搭建了属于自己博客,但由于软件与环境的更新换代，譬如Hexo、node.js等的版本变化，导致不能完全地依样画葫芦。

在执行 hexo deploy 后,出现 error deployer not found:github 的错误:
尝试了网络上许多方法都没有得到解决，如要将_config.yml配置文件中deploy的type的github改成git。

后来回忆到，在安装hexo的时候报出了这样的错误：

If you are behind a proxy, please make sure that the ‘proxy’ config is set properly.

查询问题，在nodejs npm 安装报错 无法安装 behind a proxy中找到了解决方法：

$ npm config set proxy null
